# websocket-chat-practice
Практика: WebSockets та розробка застосунків реального часу

## Структура проекту:
Структура проєкту:

Даний проєкт реалізує простий веб-чат у реальному часі на основі Node.js та WebSocket, у якому клієнти взаємодіють із сервером через постійне двостороннє з’єднання.
 - WebSocket-сервер на Node.js (бібліотека ws)
 - HTML-клієнт для обміну повідомленнями з сервером
 - Підтримка декількох користувачів одночасно
 - Відображення підключення / відключення користувачів у чаті
 - Автоматичне перепідключення при розриві зʼєднання
 - Обробка помилок WebSocket на клієнті та сервері
